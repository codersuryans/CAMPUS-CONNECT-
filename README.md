# 📦 CampusCrate

CampusCrate is a centralized "Lost & Found" platform built with the MERN stack (MongoDB, Express, React, Node.js) specifically designed for college campuses. It provides a reliable directory for students to search for lost items and securely return found belongings to their rightful owners.

## ✨ Features
1. **Lost & Found Postings**: Seamlessly report items you've lost or found.
2. **Integrated Search**: Discover matching listings using location maps and categories.
3. **Smart Claiming System**: Securely claim an item by answering specific ownership questions before communication occurs.
4. **Vibrant UI**: Enjoy an interactive, modern user interface incorporating glassmorphism layouts and responsive design.

## 🧰 Tech Stack
* **Frontend**: React (Vite), React Router, Vanilla CSS Modules
* **Backend**: Node.js, Express.js
* **Database**: MongoDB (Mongoose)

## 🚀 Quick Start
### 1. Backend
Navigate to the root directory, install dependencies, and run the server. Ensure that you have a `.env` configured with your `MONGO_URI`.
```bash
npm install
node server.js
```
The server will run on port `5000`.

### 2. Frontend
Navigate to the `frontend` folder, install dependencies, and run Vite.
```bash
cd frontend
npm install
npm run dev
```
